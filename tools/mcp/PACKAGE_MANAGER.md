# Package Manager

This project uses **pnpm**, not npm.

## Installation

```bash
pnpm install
```

## Available Scripts

```bash
pnpm start      # Start the MCP server
pnpm dev        # Start with auto-reload
pnpm format     # Format code
pnpm lint       # Lint code
```

## Why pnpm?

- Faster installation
- Better dependency management
- More efficient disk usage
- Stricter dependency resolution
